## SOC Labs — Hands-On Cybersecurity Documentation

A portfolio of hands-on SOC analyst labs covering real-world attack detection, log analysis, network forensics, and threat hunting.

## About This Repository

This repository documents practical cybersecurity exercises designed to simulate the day-to-day responsibilities of a SOC analyst. Each module covers a specific detection domain, using real tools and attack scenarios to build investigative skills.
Every lab includes:

Full step-by-step documentation
Screenshots of key findings
Technical analysis linking attack patterns to indicators
Lessons learned from a SOC analyst perspecti
